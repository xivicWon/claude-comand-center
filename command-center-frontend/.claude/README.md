# Claude Code Configuration

This directory contains development guidelines and rules for the Command Center Frontend project.

## Files

### `rules.md`
Development rules and best practices including:
- Code quality workflow
- TypeScript type safety rules
- Development checklist
- Error prevention guidelines

## Usage

These rules are automatically applied when using Claude Code for development assistance. They ensure:

- ✅ Type-safe code
- ✅ Production-ready builds
- ✅ Consistent coding standards
- ✅ Error prevention

## Quick Reference

After completing any feature:

```bash
# 1. Type check
npm run type-check

# 2. Production build
npm run build
```

For detailed guidelines, see [rules.md](./rules.md)
